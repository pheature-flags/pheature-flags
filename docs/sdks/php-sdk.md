> [Checkout official repository](https://github.com/pheature-flags/php-sdk)

## Installation guide

Install it using [composer package manager](https://getcomposer.org/download/).

```bash
composer require pheature/sdk
```

```php
<?php

declare(strict_types=1);

echo "Hollo world!";
```
